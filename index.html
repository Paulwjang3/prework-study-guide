HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • a herf="website" :used to create a hyperlink
  • img src="link of image" :used to put a image on the website, alt="message if image in broken".

CSS

  • link href="styles/style.css" rel="stylesheet" must be added to index.html
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * is a special wild card symbol that declares that all elements will have the same style properties.
  • Selector is the HTML element name at the start of the ruleset or rule.
  • Apart from the selector, each ruleset must be wrapped in curly braces. ({})
  • Declaration is a single rule like color: red;
  • Within each declaration, you must use a colon (:) to separate the property from its value or values.
  • Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
  • # is used for id
  • . is used for class

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: checks what branch we are currently on
  • git add -A: preparing for upload
  • git commit -m "message": proposal
  • git push origin main: marriage

JavaScript

  • Ctrl+Shift+I - opens Chrome DevTools
  • console.log() is amethod outputs a message to the web console
  • array is a group of related data [ ]
  • === is a Strict equality operator
  • /* */ - comment out (makes all lines of code inside inactive)
  • function is simply a set of instructions that tells the computer how to perform a certain task.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
